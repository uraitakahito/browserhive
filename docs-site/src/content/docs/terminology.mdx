The definitions on this page are **not written by hand**. They are
**extracted at build time** from `@glossary` tags (and `@category`) attached
to browserhive's TypeScript source. Fix the comment in the code and this
page follows automatically on the next build.

The "location" column of each row points at the defining `file:line`. When
in doubt about what a term means, reading that TSDoc at the source is the
shortest path.

:::note
The extracted definitions currently come from Japanese TSDoc comments in the
source code, so the table below is in Japanese. Translating the `@glossary`
comments at the source is a separate task.
:::
